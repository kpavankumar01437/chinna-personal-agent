# DevPilot AI Judging Script

## Five-Minute Demo

1. Open the dashboard.
2. Click **Trigger Sample Incident**.
3. Show the incident is detected with severity and initial status.
4. Click **Run Agents**.
5. Show the Live Agent Timeline moving through analysis, memory lookup, research, code repair, tests, security review, PR writing, and voice briefing.
6. Highlight that attempt 1 fixes only the import issue and tests still fail.
7. Highlight Mistakes & Resolutions showing self-correction.
8. Show attempt 2 fixes the API contract and validation.
9. Show final passing tests, code diff, rollback plan, time saved, confidence, memory entry, and PR preview.
10. Play the voice briefing.

## Judge-Friendly Pitch

DevPilot AI is an autonomous multi-agent DevOps teammate. It watches software failures, reasons through root causes, repairs code in a sandbox, verifies the fix, documents mistakes and resolutions, creates a rollback plan, estimates time saved, stores incident memory, and prepares a pull request for human approval.
