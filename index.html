Perception

Sees only when awake

Chinna listens for Hey Chinna WakeUp, captures local screenshots, extracts OCR text, detects active windows, and stores observations in a private vault.

Reasoning

Plans before acting

Commands are classified into safe, uncertain, approval-required, and blocked actions before execution. DevPilot uses specialized agents for repair planning and validation.

Action

Acts with guardrails

The app can open apps, search the web, observe the screen, prepare messages/calls/payments, run tests, repair sample code, and generate PR previews.

Why This Page Is Static

The full app cannot run directly on GitHub Pages because it needs a local FastAPI backend, microphone access, Windows desktop automation, OCR, Ollama, and a private local vault.

For privacy and screen-control safety, the real dashboard runs locally at http://127.0.0.1:5173 after setup.

Quick Start

For GitHub-hosted execution, open Codespaces. For full private desktop control, clone the repository and run the local Windows app.

  • Open https://codespaces.new/kpavankumar01437/chinna-personal-agent?quickstart=1
  • Download installer from https://github.com/kpavankumar01437/chinna-personal-agent/releases/latest
  • git clone https://github.com/kpavankumar01437/chinna-personal-agent.git
  • powershell -ExecutionPolicy Bypass -File .\scripts\start-chinna-local.ps1
  • powershell -ExecutionPolicy Bypass -File .\scripts\setup-omnipilot.ps1
  • cd backend; .\.venv\Scripts\python.exe -m uvicorn app.main:app --host 127.0.0.1 --port 8000
  • cd frontend; npm install; npm run dev
